NVGT.zip

This little website allows you to easily download the latest version of NVGT for any platform.

  • Download NVGT for any supported platform by simply adding it to the end of the URL. For example, you could download NVGT for macOS (latest version) with https://nvgt.zip/mac. A complete list of available platforms is below.
  • The complete list of supported platforms are: android, linux, , windows.
  • Use /dev/platform to download the latest development installer for that platform.
  • Use commits.html or commits.txt to see NVGT's latest commits.
  • Use /version.json to get the NVGT version returned back to you in JSON format, or /version.txt to get it returned in plaintext.

NVGT.zip was written by BrailleScreen and other contributors. You can view or submit improvements to the source code on GitHub.
