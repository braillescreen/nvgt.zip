NVGT.zip

This little website allows you to download the latest version of NVGT for any platform.

Here are some available options:

  • Download NVGT for any supported platform by simply adding it to the end of the URL. For example, you could download NVGT for macOS (latest version) with https://nvgt.zip/mac.
  • The supported platforms are: linux, mac, windows.
  • You can also use /version.json to get the NVGT version returned back to you in JSON format.
